# 學生會法規　可讀版 MD ＋ 結構化資料（第20屆・現行版）

> 由 `01｜公開檢閱資料/…/20.第二十屆學生議會學生自治法規/` 底下 38 份 PDF **自動解析**產生。
> 用途有二：(1) 人類**更好閱讀**；(2) 作為未來網站的**種子資料（seed data）**，可直接灌資料庫。

## 內容物

| 檔案 | 說明 |
|---|---|
| `每份法規MD/` | 38 份法規，每份一個乾淨 Markdown。含：法規名、編號、類別、現行版本日期、可展開的**修正沿革**、章條款項。 |
| `現行法規合集-第20屆.md` | 全部 38 份合成一份，最上面有分類目錄，適合全文瀏覽 / Ctrl-F 搜尋。 |
| `法規結構化-第20屆.json` | 機器可讀。每條法規 → 章 → 條（含條名、款項、所屬章、修正沿革）。這是日後 API / 全文檢索 / RAG / 時程抽取的原料。 |

## 規模

- 38 份法規、733 條
- 最高章程 3 份 / 77 條・行政 20 份 / 318 條・立法 10 份 / 203 條・司法 1 份 / 25 條・選舉 4 份 / 110 條

## JSON 欄位

```
file, category, number, name, title, current_date, current_type, session_label,
preamble, amendment_history[{date, roc, session_term, action, text}],
chapters[{title}], articles[{number, name, chapter, items[]}], article_count
```

## 重要提醒

- 這是**自動解析**的結果，已做中文去空格、沿革折行修復、款項辨識等清理，但**法律效力仍以原始 PDF 為準**。上線前建議人工校對一輪（特別是條號、金額、日期、期限數字）。
- 解析器：`parse_laws.py`（可重跑，支援其他屆別，只要換輸入資料夾）。
